## コスト最適化ポイント

### 採用した最適化

1. **VPC レス構成**: NAT Gateway ($32/月) を回避するため、Lambda を VPC 外で実行
2. **DynamoDB オンデマンド**: 低トラフィック時はプロビジョンドよりオンデマンドが安価
3. **S3 ライフサイクル**: 90日経過した画像は S3 IA に移行（ストレージ費用 40% 削減）
4. **CloudFront キャッシュ**: 画像配信は S3 直接ではなく CDN 経由（転送コスト削減）

### 回避したコストトラップ

| サービス | トラップ | 回避策 |
|---------|---------|--------|
| NAT Gateway | $0.045/時 ≒ $32/月 | VPC レス構成で回避 |
| Elastic IP | 未使用時 $0.005/時 | 使用しない |
| RDS | t3.micro でも ~$15/月 | DynamoDB で代替 |
| S3 Transfer | リージョン間転送 | 単一リージョン構成 |
| CloudWatch Logs | 大量ログ保存 | 保持期間 30日に設定 |
